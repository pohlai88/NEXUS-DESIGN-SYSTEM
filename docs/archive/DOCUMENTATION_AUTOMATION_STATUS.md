## Usage

### Generate All Documentation

```bash
pnpm generate:docs
```

This runs all three generators:
- API Reference (Markdown)
- Token Reference
- Component Documentation

### Generate Individual Documentation

```bash
# API Reference (requires api-docs.json)
pnpm extract:api-docs  # Generate JSON first
pnpm generate:api-reference

# Token Reference (requires tokens.json)
pnpm extract:tokens  # Generate JSON first
pnpm generate:docs  # Includes token docs

# Component Documentation
pnpm generate:docs  # Includes component docs
```

### Cleanup Documentation

```bash
# Preview what would be deleted
pnpm cleanup:docs:dry-run

# Actually delete outdated files
pnpm cleanup:docs
```

## Troubleshooting

### Docs Not Generating

1. **Check prerequisites**:
   - `dist/api-docs.json` exists (run `pnpm extract:api-docs`)
   - `dist/tokens.json` exists (run `pnpm extract:tokens`)
   - `components/react/` directory exists

2. **Check build process**:
   ```bash
   pnpm build:fast  # Build without adapters
   pnpm generate:docs  # Generate docs
   ```

### API Reference Missing

The API reference requires `dist/api-docs.json`:

```bash
pnpm extract:api-docs  # Generate JSON
pnpm generate:api-reference  # Generate markdown
```

### Token Reference Missing

The token reference requires `dist/tokens.json`:

```bash
pnpm extract:tokens  # Generate JSON
pnpm generate:docs  # Includes token docs
```
