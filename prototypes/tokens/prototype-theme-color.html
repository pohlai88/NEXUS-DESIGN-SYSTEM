Usage Guide

Copy-paste ready code examples for developers

Using CSS Variables

Direct access to color tokens in your CSS:

/* In your CSS or CSS-in-JS */
.element {
  color: var(--color-lux);
  background: var(--color-paper);
  border-color: var(--color-stroke);
}

.success-indicator { color: var(--color-success); }
.error-state { color: var(--color-error); }
.warning-alert { color: var(--color-warning); }

Using Semantic Classes

For HTML-only implementations:

<!-- Text colors -->
<div class="text-lux">Primary text</div>
<div class="text-lux-dim">Secondary text</div>
<div class="text-clay">Metadata text</div>
<div class="text-gold">Accent/Action</div>

<!-- Background colors -->
<div class="bg-paper">Card background</div>
<div class="bg-void">Main background</div>
<div class="bg-paper-2">Input/Hover state</div>
