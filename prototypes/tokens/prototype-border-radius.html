Design Tokens

CSS variables for border radius values

Neo-Analog Radius Tokens

/* Border Radius Tokens */
--radius-control: 2px;    /* Small controls (checkboxes, radio) */
--radius-card: 6px;       /* Cards, buttons, inputs */
--radius-panel: 8px;      /* Panels, sections */
--radius-modal: 12px;     /* Modals, dialogs */
--radius-full: 9999px;    /* Pills, avatars */

Using Semantic Classes

Apply border radius using utility classes:

<!-- Border radius classes -->
<div class="na-rounded-none">No radius</div>
<div class="na-rounded-xs">Extra small (2px)</div>
<div class="na-rounded-sm">Small (4px)</div>
<div class="na-rounded-md">Medium (6px)</div>
<div class="na-rounded-lg">Large (8px)</div>
<div class="na-rounded-xl">Extra large (12px)</div>
<div class="na-rounded-2xl">2X large (16px)</div>
<div class="na-rounded-3xl">3X large (24px)</div>
<div class="na-rounded-full">Fully rounded (9999px)</div>

Using CSS Variables

Direct access to radius tokens:

/* In your CSS */
.button {
  border-radius: var(--radius-card);
}

.card {
  border-radius: var(--radius-panel);
}

.modal {
  border-radius: var(--radius-modal);
}

.avatar {
  border-radius: var(--radius-full);
}

Figma → Neo-Analog Conversion

❌ rounded-[var(--radius/xs,2px)] → ✅ var(--radius-control)
❌ rounded-[var(--radius/md,6px)] → ✅ var(--radius-card)
❌ rounded-[var(--spacing/2,8px)] → ✅ var(--radius-panel)
❌ rounded-[var(--radius/full,9999px)] → ✅ var(--radius-full)
