Usage Patterns

Apply breakpoints to layout, grids, and visibility

Layout Shell

Clamp main content width per breakpoint.

Use .na-bp-lg with max-width: var(--na-bp); on layout containers.

Grid Columns

Change columns at breakpoints.

.cards {
  display: grid;
  gap: var(--spacing-4);
  grid-template-columns: repeat(1, minmax(0, 1fr));
}
@media (min-width: var(--bp-md)) {
  .cards { grid-template-columns: repeat(2, minmax(0, 1fr)); }
}
@media (min-width: var(--bp-xl)) {
  .cards { grid-template-columns: repeat(3, minmax(0, 1fr)); }
}

Visibility

Toggle UI below or above breakpoints.

.hide-mobile { display: none; }
@media (min-width: var(--bp-sm)) {
  .hide-mobile { display: inline-flex; }
}

.hide-desktop { display: inline-flex; }
@media (min-width: var(--bp-lg)) {
  .hide-desktop { display: none; }
}
