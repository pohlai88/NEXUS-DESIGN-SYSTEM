Using Semantic Classes

Apply blur using utility classes:

<!-- Blur classes -->
<div class="na-blur-none">No blur (sharp)</div>
<div class="na-blur-xs">Extra small (2px)</div>
<div class="na-blur-sm">Small (4px)</div>
<div class="na-blur-md">Medium (6px)</div>
<div class="na-blur-lg">Large (8px)</div>
<div class="na-blur-xl">Extra large (12px)</div>
<div class="na-blur-2xl">2X large (20px)</div>
<div class="na-blur-3xl">3X large (32px)</div>

Using CSS Variables

Direct access to blur tokens:

/* In your CSS */
.loading-state {
  filter: blur(var(--blur-md));
}

.privacy-mask {
  filter: blur(var(--blur-2xl));
}

/* Glassmorphism effect */
.glass-card {
  backdrop-filter: blur(var(--blur-xl));
  background: rgba(18, 18, 20, 0.7);
}

/* Transition blur on hover */
.hover-blur {
  transition: filter 0.3s var(--ease-premium);
}

.hover-blur:hover {
  filter: blur(var(--blur-lg));
}

Backdrop Filter (Glassmorphism)

Modern frosted glass effects:

/* Glassmorphism card */
.glass-card {
  backdrop-filter: blur(12px);
  background: rgba(18, 18, 20, 0.7);
  border: 1px solid rgba(255, 255, 255, 0.1);
}

/* Modal backdrop */
.modal-backdrop {
  backdrop-filter: blur(4px);
  background: rgba(0, 0, 0, 0.5);
}
