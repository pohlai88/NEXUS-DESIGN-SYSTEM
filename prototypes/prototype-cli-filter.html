How It Works

1. Overlay Technique

The input field is made transparent with a visible caret. A backdrop div behind it renders colored tokens at the exact same position.

2. Token Parsing

CLI Parser regex recognizes: keys (text:), operators (><=), values, and quotes. Each token gets a semantic type for styled highlighting.

3. Reactive UI

On every keystroke, the parser runs and the backdrop updates, creating a "live syntax highlighting" effect as the user types.
