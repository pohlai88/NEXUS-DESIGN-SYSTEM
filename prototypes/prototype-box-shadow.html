Using Semantic Classes

Apply shadows using utility classes:

<!-- Drop shadows -->
<div class="na-shadow-none">No shadow</div>
<div class="na-shadow-2xs">Extra extra small</div>
<div class="na-shadow-xs">Extra small</div>
<div class="na-shadow-sm">Small</div>
<div class="na-shadow-md">Medium (default cards)</div>
<div class="na-shadow-lg">Large (panels)</div>
<div class="na-shadow-xl">Extra large (modals)</div>
<div class="na-shadow-2xl">2X large</div>

<!-- Inset shadows -->
<div class="na-shadow-inset-2xs">Inset 2XS</div>
<div class="na-shadow-inset-xs">Inset XS</div>
<div class="na-shadow-inset-sm">Inset SM</div>

Using CSS Variables

Direct access to shadow tokens:

/* In your CSS */
.card {
  box-shadow: var(--shadow-md);
}

.modal {
  box-shadow: var(--shadow-xl);
}

.input:focus {
  box-shadow: var(--shadow-inset-sm);
}

/* Hover transitions */
.interactive-card {
  box-shadow: var(--shadow-sm);
  transition: box-shadow 0.15s var(--ease-premium);
}

.interactive-card:hover {
  box-shadow: var(--shadow-md);
}
