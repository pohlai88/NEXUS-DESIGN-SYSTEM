Usage Code

Example code for using the attractive theme with AIBOS buttons.

// Import theme
import { attractiveTheme, ThemeProvider } from '@aibos/design-system/themes';

// Use in your app
<ThemeProvider customThemes={[attractiveTheme]}>
  <button className="na-btn na-btn-primary">Primary</button>
  <button className="na-btn">Secondary</button>
  <button className="bg-accent text-accent-foreground px-6 py-3 rounded-lg">Accent</button>
</ThemeProvider>
